Contents
==============
This GSQL 101 tutorial contains 

- friendship.csv: the edge data
- person.csv: the vertex data
- gsql101.gsql: the gsql script to setup the schema, load the data, and stored procedures. 


How To Run
===============
gsql gsql101.gsql
