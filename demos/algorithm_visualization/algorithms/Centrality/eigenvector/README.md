#### Via TigerGraph CLI

```bash
$ tg box algos install <Eigenvector Algorithm>
```

#### Via GSQL terminal

```bash
GSQL > BEGIN
# Paste <Eigenvector Algorithm> code after BEGIN command
GSQL > END 
GSQL > INSTALL QUERY <Eigenvector Algorithm>
```
